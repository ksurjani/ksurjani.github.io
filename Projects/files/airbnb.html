AirBnb Big Data Analysis

  • Tech Stack: Python (Pandas, NumPy), NLP (NLTK / Text Preprocessing), Topic Modeling (LDA, Bigram/Trigram), Transformer Models (BERT, RoBERTa), LLMs (LLaMA2 via Ollama, LangChain), Reddit API (PRAW for data collection), Data Processing & Feature Engineering
  • Github URL: Project Link

Built an end-to-end text analytics pipeline to identify emerging travel and pop-culture trends for Airbnb property innovation. Collected and processed large-scale Reddit data using PRAW, and applied multiple NLP techniques including LDA, n-gram modeling, and transformer-based sentiment analysis to extract meaningful themes. Iteratively improved model performance by experimenting with advanced approaches like Seeded LDA, BERT/RoBERTa, and finally LLM-based classification using LangChain + LLaMA2 to generate structured, actionable insights for themed property recommendations.
